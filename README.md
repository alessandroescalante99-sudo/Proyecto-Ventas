# Análisis de Ventas utilizando Python y Power BI para identificar tendencias de ventas, comportamiento de productos y desempeño comercial.

## Descripción

Este proyecto consiste en el desarrollo de un Dashboard de Ventas utilizando Python para el proceso ETL y Power BI para la visualización de datos.

A partir de una base de datos comercial se realizó la limpieza, transformación e integración de la información para analizar el desempeño de las ventas desde distintas perspectivas.

## Objetivos

Analizar el rendimiento de las ventas.

Identificar los productos más vendidos.

Comparar categorías y subcategorías.

Analizar la evolución mensual.

Medir el tiempo promedio de envío.

Obtener indicadores comerciales mediante un dashboard interactivo.
